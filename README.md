# Deep Learning Approach for Classification of Eye Diseases based on Color Fundus Images using MATLAB

Final-year (RMK Engineering College, ECE) project, Batch 50 — a deep learning system that classifies eye diseases from color fundus (retinal) images, aimed at supporting earlier detection of conditions like age-related macular degeneration (AMD) and diabetic retinopathy (DR).

## Team

Venkata Gunasekhar V, M. Shaama, Vinay Krishna K N — under the guidance of Dr. Sumitra V, Dept. of Electronics & Communication Engineering, R.M.K. Engineering College.

## Tech Stack

MATLAB, deep learning (transfer learning), DenseNet architecture, image processing.

## Abstract

Eye disease recognition typically requires years of clinical experience. This project explores a deep-learning-based approach to predicting eye diseases (such as AMD and diabetic retinopathy) from color fundus images, using publicly available datasets, image pre-processing, and a DenseNet-based feature extractor combined with transfer learning for classification.

**Pipeline / modules:**
1. Data collection (public eye-disease fundus datasets)
2. Pre-processing
3. Feature extraction (DenseNet)
4. Training
5. Testing
6. Classification

**Motivation:** conditions like AMD (macular degeneration) and diabetic retinopathy can lead to vision loss or blindness if not caught early. The goal is a tool that assists clinicians in earlier, more consistent detection — not replace them.
